Welcome to Project Fuse.

The UX team has created a common CCW framework using Bootstrap. We have customized the components provided by Bootstrap to build our CCW pages with the correct styling, layouts, and different components. These templates are meant to bring value for our teams to promote consistency, decrease coding efforts, and reduce page rework. This page provides one place for the framework and components to be housed and downloaded as engineering builds the pages.

Our Templates

The templates can be found on this page and can be downloaded as a ZIP file. We have consolidated all needed UI components. This Zip File contains our resultant framework. It is advised to fork off from this framework and recommit here when more developments are made while building CCW.
